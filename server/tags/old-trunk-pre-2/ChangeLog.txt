1.8.2
=====
o Bug fixes and stability enhancements.
o Enhanced initialization of BlueFish cards.
o Support for 16 bits embedded audio on BlueFish cards.
o Video files is proritiesed over tga if the filename is the same.
o Support for HD on DecLink cards.
o External key is enabled by default on DecLink cards.
o Support for playing compressed template file, .ct.
